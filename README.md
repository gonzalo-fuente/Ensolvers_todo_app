# Running the Backend

In one terminal type:

### `cd todo_backend`

### `npm install`

### `npm run create`

### `npm run migrate`

### `npm run add-seed`

### `npm start` - runs the server on http://localhost:5000

# Running the Frontend

In another terminal type:

### `cd todo_frontend`

### `npm install`

### `npm run serve` - runs the server on http://localhost:8080
